# Test Suite 001: AI Industry and Social Media

This suite tests entity extraction for modern tech industry content with:

- Social media handles (@username)
- Stock symbols ($TICK)
- Organization name changes (X/Twitter)
- Professional titles with organizations
- Product versions (GPT-5, Claude 3)
- Research organizations and sub-organizations
- Location hierarchies
- Hashtag topics
- Multiple references to same entity with different forms

Particular focus on:
- Company/product/person disambiguation
- Handle/canonical name mapping
- Organizational relationships
- Geographic entity resolution
- Title parsing
